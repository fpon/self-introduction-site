# Self Introduction Site

自己紹介・ポートフォリオサイトです。

## About

このサイトは、自分自身のことを発信するための個人サイトです。

### コンテンツ

- **経歴** - これまでやってきたこと、経験してきたこと
- **制作物** - 個人開発プロジェクトやポートフォリオ
- **思想** - 考えていること、大切にしていること
- **マネジメント資料** - マネジメント学習のアウトプットや資料

## Tech Stack

- **Framework**: Next.js (Static Site Generation) / React
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui
- **Package Manager**: Bun
- **Linter / Formatter**: Biome
- **Hosting**: Vercel（予定）
